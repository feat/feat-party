页面的内容区块

## 何时使用

页面上展示多种信息时，需要进行区块分类。

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| title | 区块标题 | string | |
| subHeader | 区块二级标题，在区块头部上展示说明或者，设置按钮 | node | |
| secondary | 二级区块，改变样式 | boolean | `false` |
| namespace | 命名空间 | string | `ft` |
