国旗。

## 何时使用

- 显示各国国旗；

## API

### DigiInput

如正常的i， 可输入其他参数

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| country | 国家代号或名字 | string  | - |

国家编码规格同[semantic flag](https://semantic-ui.com/elements/flag.html) 组件一致.
