占位符。

## 何时使用

- 开发时使用， 未开发模块先占位；

## API

### Loader

可输入div所有的参数

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| width | 样式 | number  | `100%` |
| ratio | 宽高比例 | number  | - |
| text | 说明所占位置功能 | string  | - |
