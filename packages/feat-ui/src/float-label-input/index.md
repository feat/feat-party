浮动标签输入框。

## 何时使用

- 在表单中使用, 横向位置不足时。

## API

### float-label-input
拥有`Input`组件所有的参数

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| label | 浮动标签 | string  | - |
| labelStyle | 控制标签的样式 | object | - |
| labelClassName | 控制标签的类名 | string | - |
